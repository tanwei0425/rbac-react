# rbac-react 权限管理系统

## rbac-reac 配合 rbac-egg 简单上手，可扩展性高

一个可用于生产环境的基于 react、eggjs 的 rbac权限管理系统。

线上地址：[预览](https://admin.hellotanwei.cn/login)

## 技术栈

- React17（hooks、class）
- React-router 6x
- Redux 4x
- Ant Design 4x
- Ahooks
- Less
- Axios
- Socket.io
- Mock
- Lodash
- Braft-editor
